# Copiar antes de que desaparezca
## Código PHP
```PHP
<?php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
echo $_FILES["fileToUpload"]["tmp_name"];

move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file);
sleep(10);
?>
```
## Código en PowerShell
```PowerShell
cd C:\xampp\tmp

for(1)
{
    mv * ..\supersecreto
}
```
