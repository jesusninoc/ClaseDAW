### Acceder a MySQL como root

### Crear usuario en MySQL
```MySQL
CREATE USER 'usuario'@'localhost' IDENTIFIED BY 'passwordsecreto';

GRANT ALL PRIVILEGES ON *.* TO 'usuario'@'localhost' WITH GRANT OPTION;

FLUSH PRIVILEGES;
```
